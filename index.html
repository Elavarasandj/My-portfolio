My Projects

GESTURE TUBE: HANDS-FREE YOUTUBE CONTROL SYSTEM

Description:
This project allows users to control YouTube video playback using hand gestures, eliminating the need for physical interaction. Built using Python, OpenCV, and MediaPipe, it detects real-time hand movements via a webcam and maps them to YouTube controls using PyAutoGUI.
Key Features:
1.Play/Pause videos with open-palm gesture
2.Control volume with up/down finger gestures
3.Skip forward or backward using swipe gestures
4.Fully hands-free experience using only a webcam

FAKE IMAGE DETECTION USING CNN ALOGORITHM

Description:
This project aims to detect whether an image has been digitally manipulated or is generated by AI (deepfakes or GANs). It uses image processing techniques and machine learning models to analyze patterns and inconsistencies that help identify fake images.
Key Features:
1.Classifies images as real or fake
2.Uses convolutional neural networks (CNN) for deep feature extraction
3.Detects common manipulations (GAN-generated, deepfakes, photoshopped)
4.User-friendly interface for image upload and prediction
